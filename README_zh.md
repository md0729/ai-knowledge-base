# 🧠 AI 知識庫（AI Knowledge Base）

本專案用於建立一套結構化的 AI 與系統整合知識庫，將零散的學習內容與專案經驗轉化為可重複使用的工程知識。

---

## 🎯 目的

本知識庫的目標在於：

* 累積技術知識（而非一次性查詢）
* 建立有結構的工程理解
* 記錄實際專案經驗與除錯歷程
* 支援論文撰寫與面試表達

---

## 🧱 專案結構

```id="nx76nd"
/wiki_core      → 核心知識（模型、方法、系統設計）
/projects       → 專案經驗（問題、除錯、解法）
/raw            → 原始資料（文章、靈感、筆記）
```

---

## 📌 知識分類原則

### 🧠 wiki_core（核心知識）

* 可重複使用的概念
* 抽象化的技術知識
* 系統設計方法

範例：

* LSTM window size 設計
* YOLO + LSTM 多模態融合
* 人機協作流程設計

---

### 🔧 projects（實戰經驗）

* 實際開發過程中的問題
* 除錯紀錄
* 系統整合經驗

範例：

* socket 斷線問題
* MediaPipe 偵測不穩
* 模型整合問題

---

### 🗑 raw（原始資料）

* 尚未整理的資訊
* 可快速記錄，不需結構

---

## 🔄 使用流程

```id="c93k0r"
輸入（問題 / 想法 / 文章）
→（可選）存入 raw
→ 使用 AI 整理
→ 存入 wiki_core 或 projects
```
